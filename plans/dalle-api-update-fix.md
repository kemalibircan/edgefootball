# DALL-E 3 Görsel Oluşturma Sorunu - Güncel API Çözümü

## 🔍 Sorun Analizi

**Durum**: OpenAI API key var ve chat çalışıyor, ancak görsel oluşturma çalışmıyor.

**Olası Nedenler**:
1. ❌ DALL-E 3 API endpoint'i değişmiş olabilir
2. ❌ Request formatı güncel değil
3. ❌ Response formatı değişmiş
4. ❌ Timeout süresi yetersiz
5. ❌ API key'in DALL-E erişimi yok

## ✅ Güncel DALL-E 3 API (2024+)

### OpenAI API Değişiklikleri

**Eski Format** (2023):
```python
response = await client.post(
    "https://api.openai.com/v1/images/generations",
    headers={
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json",
    },
    json={
        "model": "dall-e-3",
        "prompt": prompt,
        "n": 1,
        "size": "1792x1024",
        "quality": "hd",
        "style": "vivid",
    }
)
```

**Yeni Format** (2024+):
```python
from openai import AsyncOpenAI

client = AsyncOpenAI(api_key=api_key)

response = await client.images.generate(
    model="dall-e-3",
    prompt=prompt,
    size="1792x1024",
    quality="hd",
    style="vivid",
    n=1,
)
```

### Adım 2: Image Generation Modülü Güncelleme

**Dosya**: `app/image_generation.py`

```python
from __future__ import annotations

import asyncio
import hashlib
from datetime import datetime, timezone
from pathlib import Path
from typing import Any, Dict, List, Optional
import logging

import httpx
from openai import AsyncOpenAI, OpenAIError

from app.config import Settings, get_settings

logger = logging.getLogger(__name__)


class ImageGenerationError(Exception):
    pass


async def generate_football_slider_image(
    prompt: str,
    settings: Settings,
    size: str = "1792x1024",
    quality: str = "hd",
    style: str = "vivid",
) -> Dict[str, Any]:
    """
    Generate a single image using OpenAI DALL-E 3 (Updated 2024 API).
    
    Args:
        prompt: The image generation prompt
        settings: App settings with OpenAI API key
        size: Image size (1024x1024, 1792x1024, or 1024x1792)
        quality: Image quality (standard or hd)
        style: Image style (vivid or natural)
    
    Returns:
        Dict with generated image metadata
    """
    if not settings.openai_api_key:
        error_msg = "OpenAI API key not configured. Set OPENAI_API_KEY in .env"
        logger.error(error_msg)
        raise ImageGenerationError(error_msg)
    
    # Validate size
    valid_sizes = ["1024x1024", "1792x1024", "1024x1792"]
    if size not in valid_sizes:
        logger.warning(f"Invalid size {size}, using 1792x1024")
        size = "1792x1024"
    
    # Validate quality
    if quality not in ["standard", "hd"]:
        logger.warning(f"Invalid quality {quality}, using hd")
        quality = "hd"
    
    # Validate style
    if style not in ["vivid", "natural"]:
        logger.warning(f"Invalid style {style}, using vivid")
        style = "vivid"
    
    try:
        logger.info(f"Generating DALL-E 3 image: size={size}, quality={quality}, style={style}")
        logger.debug(f"Prompt: {prompt[:100]}...")
        
        # Initialize OpenAI client with updated SDK
        client = AsyncOpenAI(
            api_key=settings.openai_api_key,
            timeout=180.0,  # 3 minutes timeout
            max_retries=2,
        )
        
        # Generate image using new SDK
        response = await client.images.generate(
            model="dall-e-3",
            prompt=prompt,
            size=size,
            quality=quality,
            style=style,
            n=1,
            response_format="url",  # Get URL instead of b64_json
        )
        
        if not response.data or len(response.data) == 0:
            raise ImageGenerationError("No image generated by DALL-E")
        
        image_data = response.data[0]
        image_url = image_data.url
        
        if not image_url:
            raise ImageGenerationError("No image URL in DALL-E response")
        
        logger.info(f"DALL-E image generated successfully: {image_url[:50]}...")
        
        # Download the generated image
        async with httpx.AsyncClient(timeout=60.0) as http_client:
            image_response = await http_client.get(image_url)
            
            if image_response.status_code != 200:
                raise ImageGenerationError(
                    f"Failed to download generated image: {image_response.status_code}"
                )
            
            image_bytes = image_response.content
        
        # Save to local storage
        static_dir = Path(__file__).resolve().parent / "static" / "slider"
        static_dir.mkdir(parents=True, exist_ok=True)
        
        timestamp = datetime.now(timezone.utc).strftime("%Y%m%d_%H%M%S")
        image_hash = hashlib.md5(image_bytes).hexdigest()[:8]
        filename = f"dalle_{timestamp}_{image_hash}.png"
        local_path = static_dir / filename
        
        with open(local_path, "wb") as f:
            f.write(image_bytes)
        
        relative_path = f"/static/slider/{filename}"
        
        logger.info(f"Image saved to: {relative_path}")
        
        # Get revised prompt if available
        revised_prompt = getattr(image_data, 'revised_prompt', prompt)
        
        return {
            "url": image_url,
            "local_path": str(local_path),
            "relative_url": relative_path,
            "prompt": prompt,
            "revised_prompt": revised_prompt,  # DALL-E may revise the prompt
            "metadata": {
                "model": "dall-e-3",
                "size": size,
                "quality": quality,
                "style": style,
                "generated_at": datetime.now(timezone.utc).isoformat(),
                "file_size_bytes": len(image_bytes),
            },
        }
        
    except OpenAIError as e:
        error_msg = f"OpenAI API error: {str(e)}"
        logger.error(error_msg, exc_info=True)
        raise ImageGenerationError(error_msg)
    
    except httpx.TimeoutException as e:
        error_msg = f"Request timeout: {str(e)}"
        logger.error(error_msg)
        raise ImageGenerationError(error_msg)
    
    except Exception as e:
        error_msg = f"Unexpected error during image generation: {str(e)}"
        logger.error(error_msg, exc_info=True)
        raise ImageGenerationError(error_msg)


async def generate_slider_images_batch(
    count: int = 3,
    settings: Optional[Settings] = None,
) -> List[Dict[str, Any]]:
    """
    Generate multiple slider images in batch.
    """
    if settings is None:
        settings = get_settings()
    
    if not settings.openai_api_key:
        error_msg = "OpenAI API key not configured"
        logger.error(error_msg)
        raise ImageGenerationError(error_msg)
    
    prompts = [
        "Modern minimalist football stadium at night with dramatic neon green lighting, navy blue sky, futuristic sports aesthetic, ultra high quality, cinematic",
        "Abstract football tactics visualization with glowing neon green lines on navy blue background, data-driven sports concept, geometric patterns, modern design",
        "Cinematic shot of football on pitch with AI holographic overlays, neon green accents, navy atmosphere, cutting-edge technology, dramatic lighting",
    ]
    
    count = min(count, len(prompts))
    
    logger.info(f"Starting batch generation of {count} images")
    
    tasks = []
    for i in range(count):
        prompt = prompts[i]
        tasks.append(generate_football_slider_image(prompt, settings))
    
    results = await asyncio.gather(*tasks, return_exceptions=True)
    
    generated_images = []
    errors = []
    
    for i, result in enumerate(results):
        if isinstance(result, Exception):
            error_msg = f"Image {i+1} generation failed: {str(result)}"
            logger.error(error_msg)
            errors.append(error_msg)
            continue
        generated_images.append(result)
    
    if not generated_images:
        error_detail = "\n".join(errors) if errors else "Unknown error"
        raise ImageGenerationError(f"Failed to generate any images:\n{error_detail}")
    
    if errors:
        logger.warning(f"{len(errors)} image(s) failed, but {len(generated_images)} succeeded")
    
    logger.info(f"Batch generation completed: {len(generated_images)} images generated")
    
    return generated_images


async def generate_match_based_slider_images(
    settings: Optional[Settings] = None,
) -> List[Dict[str, Any]]:
    """
    Generate slider images based on today's matches.
    """
    if settings is None:
        settings = get_settings()
    
    if not settings.openai_api_key:
        error_msg = "OpenAI API key not configured"
        logger.error(error_msg)
        raise ImageGenerationError(error_msg)
    
    from app.fixture_board import get_fixture_board_page
    
    try:
        logger.info("Fetching today's fixtures for match-based slider generation")
        
        fixtures_payload = get_fixture_board_page(
            settings=settings,
            page=1,
            page_size=10,
            target_date=None,
            sort="desc",
            game_type="all",
            featured_only=False,
        )
        
        fixtures = fixtures_payload.get("items", [])[:3]
        
        if not fixtures:
            logger.warning("No fixtures found, falling back to default prompts")
            return await generate_slider_images_batch(count=3, settings=settings)
        
        logger.info(f"Found {len(fixtures)} fixtures for slider generation")
        
        prompts = []
        for idx, fixture in enumerate(fixtures):
            home_team = fixture.get("home_team_name", "Team A")
            away_team = fixture.get("away_team_name", "Team B")
            league = fixture.get("league_name", "Football League")
            
            match_result = fixture.get("markets", {}).get("match_result", {})
            home_odd = match_result.get("home", 2.0)
            away_odd = match_result.get("away", 2.0)
            
            logger.info(f"Creating prompt for match {idx+1}: {home_team} vs {away_team}")
            
            prompt = (
                f"Professional sports betting advertisement banner featuring: "
                f"'{home_team} vs {away_team}' match from {league}. "
                f"Modern minimalist design with navy blue gradient background. "
                f"Neon lime green glowing accents and text. "
                f"Include football stadium atmosphere, dramatic lighting. "
                f"Add subtle betting odds display: {home_odd:.2f} and {away_odd:.2f}. "
                f"Futuristic AI-powered sports analytics aesthetic. "
                f"Clean, professional, cinematic quality, ultra high definition. "
                f"Text overlay: 'EdgeFootball AI Predictions'."
            )
            prompts.append(prompt)
        
        logger.info(f"Starting parallel generation of {len(prompts)} match-based images")
        
        tasks = [generate_football_slider_image(prompt, settings) for prompt in prompts]
        results = await asyncio.gather(*tasks, return_exceptions=True)
        
        generated_images = []
        errors = []
        
        for i, result in enumerate(results):
            if isinstance(result, Exception):
                error_msg = f"Match-based image {i+1} failed: {str(result)}"
                logger.error(error_msg)
                errors.append(error_msg)
                continue
            generated_images.append(result)
        
        if not generated_images:
            logger.error("All match-based images failed, falling back to default prompts")
            return await generate_slider_images_batch(count=3, settings=settings)
        
        if errors:
            logger.warning(f"{len(errors)} match-based image(s) failed, but {len(generated_images)} succeeded")
        
        logger.info(f"Match-based generation completed: {len(generated_images)} images")
        
        return generated_images
        
    except Exception as e:
        logger.error(f"Failed to generate match-based images: {e}", exc_info=True)
        logger.info("Falling back to default prompts")
        return await generate_slider_images_batch(count=3, settings=settings)


def get_default_slider_prompts() -> List[str]:

    ]
```

## 🔍 Hata Ayıklama

### Hata 1: "OpenAI API key not configured"

**Çözüm**:
```bash
# .env dosyasına ekle
echo "OPENAI_API_KEY=sk-your-key-here" >> .env

# Docker'ı yeniden başlat
docker-compose restart app
```

### Hata 2: "Invalid API key"

**Çözüm**:
```bash
# API key'i test et
curl https://api.openai.com/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"

# Yeni key oluştur: https://platform.openai.com/api-keys
```

### Hata 3: "You exceeded your current quota"

**Çözüm**:
- OpenAI hesabına kredi ekle
- Billing sayfasını kontrol et: https://platform.openai.com/account/billing

### Hata 4: "Timeout"

**Çözüm**:
```python
# timeout'u artır
client = AsyncOpenAI(
    api_key=settings.openai_api_key,
    timeout=300.0,  # 5 dakika
)
```
