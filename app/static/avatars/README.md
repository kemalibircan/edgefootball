# Avatar Assets

Bu klasordeki avatar PNG dosyalari [DiceBear Open Peeps](https://www.dicebear.com/styles/open-peeps) kaynagindan uretilmistir.

- Lisans: [CC0-1.0](https://www.dicebear.com/licenses/)
- Uretim API: `https://api.dicebear.com/9.x/open-peeps/png?seed=<SEED>&size=256`
